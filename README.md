# K-Modes Clustering Analysis

## Setup

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Add your data:
- Create a `data/` folder
- Add your CSV file as `raw_data.csv`

## Run the Analysis

Run these notebooks in order:

1. `preprocessing.ipynb` - Clean data
2. `clustering.ipynb` - Cluster data
3. `clustering_analysis.ipynb` - Analyze results
4. `visualizations.ipynb` - Create visualizations
